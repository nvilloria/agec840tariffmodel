AGEC 840 Tariff Model

Nelson Villoria

Overview

This vignette implements a one-country, one-commodity partial equilibrium trade policy model based on Francois (1997)’s Perfect Substitutes Trade Model and extends it to cover import tariffs, production subsidies, consumption subsidies, and binding import quotas.

Model Equations

  • Domestic demand: \(Q_d = K_d \times P_c^{\eta_d}\) (where \(P_c\) is the consumer price)
  • Domestic supply: \(Q_s = K_s \times P_p^{\epsilon_s}\) (where \(P_p\) is the producer price)
  • Import supply: \(M_s = K_{ms} \times P_w^{\epsilon_{ms}}\) (where \(P_w\) is the world price)
  • Import demand: \(M_d = Q_d - Q_s\)
  • Market clearing: \(M_d = M_s\) (solved for \(P_d\))

Price linkages: * \(P_d = P_w \times (1 + \tau)\) [tariff wedge] * \(P_c = P_d \times (1 - sc)\) [consumption subsidy lowers consumer price] * \(P_p = P_d \times (1 + sp)\) [production subsidy raises producer price]

Worked Example: Stylized “Steel” Benchmark

This example uses the stylized “steel” benchmark from Francois (1997).

Step 1: Calibration

We recover the structural constants \(K_d, K_s, K_{ms}\) and the derived excess-demand elasticity \(\eta_{md}\) from benchmark data.

library(agec840tariffmodel)
cal <- calibrate_model(
  kd     = 1000,   # benchmark domestic consumption
  ks     =  700,   # benchmark domestic production
  eta.d  =  -0.30, # demand elasticity
  eps.s  =   0.50, # supply elasticity
  eps.ms =   5.00, # import supply elasticity
  tau    =   0.10  # benchmark tariff: 10%
)

cat(sprintf("  K_d    = %.4f\n", cal$k.d))
#>   K_d    = 1000.0000
cat(sprintf("  K_s    = %.4f\n", cal$k.s))
#>   K_s    = 700.0000
cat(sprintf("  K_ms   = %.4f\n", cal$k.ms))
#>   K_ms   = 483.1530
cat(sprintf("  eta.md = %.4f  (excess demand elasticity)\n", cal$eta.md))
#>   eta.md = -2.1667  (excess demand elasticity)

Experiment 5: Binding Import Quota

We restrict imports to 80% of the benchmark level (240 units). The welfare impact depends heavily on who captures the quota rents.

Price Transmission and Country Size

The price transmission elasticity (\(d \log P_d / d \log(1+\tau)\)) indicates how much of a tariff change is passed through to domestic prices. It approaches 1 for a small country (infinite import supply elasticity) and falls toward 0 as the country becomes larger.

eps.grid <- c(0.5, 1, 2, 4, 8, 20, 100, 1000)
trans    <- sapply(eps.grid, function(e) {
  cal.tmp <- calibrate_model(kd=1000, ks=700, eta.d=-0.30, eps.s=0.50, eps.ms=e, tau=0.10)
  linear_approx(cal.tmp, tau=0.20)$transmission.domestic
})
res <- data.frame(eps.ms = eps.grid, transmission = trans)
knitr::kable(res, digits = 4, col.names = c("Import Supply Elasticity (eps.ms)", "Price Transmission Elasticity"))
